# 📑 [문서 1] LLM 모델 비교·선정 보고서

### 1. 비교 목적

* 비매치(BeMatch) 서비스의 젠지(Gen-Z) 세대 동호인 타겟 SNS 마케팅 콘텐츠 기획 자동화 과업에 가장 적합한 트렌디한 한국어 표현력, 지정된 플랫폼 출력 형식 준수 능력, 시각적 레이아웃 구상력을 갖춘 최적의 대규모 언어 모델(LLM)을 선정하고자 함.

### 2. 비교에 사용한 동일 입력 (테스트 프롬프트)

> "배드민턴 초보자들이 자주 실수하는 '서브 규정(서비스 폴트)'에 대해 인스타그램 카드뉴스 5장짜리 초안을 작성해줘. 톤앤매너는 젠지 세대를 타겟으로 하도록 친근하고 밈 친화적으로 해줘."

3. 평가 축 및 모델별 점수표 

* **평가 축 정의:**
1. **한국어 자연스러움 (밈 표현력):** 젠지 세대가 일상적으로 사용하는 신조어나 밈 감성을 어색하지 않고 유연하게 구사하는가.

2. **형식 준수 (분량 통제):** 요청한 '5장짜리 카드뉴스 규격'과 세부 제약 사항을 정확하게 지켜 출력하는가.

3. **가독성 (시각적 레이아웃):** 인스타그램 카드뉴스 특성에 맞게 줄글을 배제하고 직관적인 디자인/비주얼 아이디어를 제시하는가.

4. **비용 효율성 (토큰 배수):** 모델의 체급 및 제공 환경 대비 업무 생산성 향상에 기여하는 효율성을 갖추었는가.





| 평가 축 | 모델 A: GPT-5 | 모델 B: Claude Sonnet 4.6 | 모델 C: Gemini 3 Flash |
| --- | --- | --- | --- |
| **1. 한국어 자연스러움** | 5 / 5 | 5 / 5 | 4 / 5 |
| **2. 형식 준수** | 5 / 5 | 5 / 5 | 5 / 5 |
| **3. 가독성** | 4 / 5 | 5 / 5 | 3 / 5 |
| **4. 비용 효율성** | 4 / 5 | 4 / 5 | 5 / 5 |
| **총점** | **18 / 20** | **19 / 20** | **17 / 20** |

**제약 사항 기록 (재현성):** 

**사용 환경:** 전 모델 웹(Web UI) 기반 텍스트 생성 파이프라인 활용.

**사용 날짜:** 2026-06-06.

**주요 설정:** Web UI 기본 설정 (Temperature 등 별도 조절 불가).

### 4. 모델별 결과 요약

#### ① 모델 A: GPT-5 (x1 토큰 배수)

* **요약:** 젠지 타겟에 대한 이해도가 매우 높음. '갑분-폴트', '폴트각', 'No cap', '현타 ON' 등 트렌디한 신조어와 밈을 적재적소에 활용함. 5장의 규격을 정확히 지켰으며 캡션과 해시태그까지 깔끔하게 마무리함. 다만 가독성 측면에서 텍스트 나열 위주의 구성을 보임.

#### ② 모델 B: Claude Sonnet 4.6 (x1 토큰 배수)

* **요약:** 기획 의도와 플랫폼 특성을 가장 완벽하게 파악함. 인스타그램 UI를 고려하여 ASCII 코드로 실제 카드 형태의 사각형 레이아웃을 직접 그려 시각적 가독성을 극대화함. "ㅇㅈ? ㅇㅇㅈ", "찐임" 같은 구어체 밈을 자연스럽게 녹여냈으며, 디자인 포인트와 색상 가이드까지 실무자가 바로 쓸 수 있는 수준으로 출력함.

#### ③ 모델 C: Gemini 3 Flash (x0.5 토큰 배수)

* **요약:** 가격 대비 성능(0.5배수)과 속도가 매우 뛰어남. '민턴이', '킹받음', '폼 미쳤다' 같은 대중적인 밈을 적절히 매칭하고 5장 분량을 준수함. 다만 '튜터의 팁'과 같은 부가적인 격려 문구가 포함되어 카드뉴스 대본 자체의 집중도가 다소 분산되며, 시각적 레이아웃 구상안이 비교 모델에 비해 평이함.

5. 최종 선정 결론 (3줄 이상 근거 필수 준수) 

비매치의 Z세대 타겟 SNS 콘텐츠 기획 자동화 과업에서는 텍스트의 세련된 톤앤매너뿐만 아니라 디자이너나 작업자가 즉시 인지할 수 있는 시각적 레이아웃 포맷팅 능력이 최우선입니다. Claude Sonnet 4.6은 단순 텍스트 출력을 넘어 실제 카드뉴스 규격에 맞춘 사각 프레임 형태의 레이아웃을 직관적으로 제시하여 가독성을 압도적으로 끌어올렸습니다. 또한 Z세대의 감성을 저해하지 않는 선에서 비주얼 포인트와 디자인 컬러 가이드라인까지 구체적으로 제안했기에, 업무 자동화 효율이 가장 높은 **Claude Sonnet 4.6을 최종 모델로 선정**합니다.
